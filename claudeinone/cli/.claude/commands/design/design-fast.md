# Design Fast

## Invocation
`/design:fast [desc]`

## Purpose
Deliver a functional UI prototype quickly.

## Agent Routing
- ui-ux-designer
- frontend-developer

## Execution Protocol
1. Define core user flow.
2. Build functional UI with minimal polish.
3. Ensure responsive basics and form validation.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Implemented UI files
2. Known design debt
