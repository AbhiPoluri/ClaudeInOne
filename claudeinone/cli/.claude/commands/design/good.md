# Design Good

## Invocation
`/design:good [brief]`

## Purpose
Deliver production-grade polished UX with accessibility and performance.

## Agent Routing
- ui-ux-designer
- frontend-developer
- accessibility-auditor

## Execution Protocol
1. Run visual/UX research pass.
2. Define design system choices.
3. Implement high-fidelity responsive UI.
4. Add accessibility and motion standards.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Design implementation
2. UX rationale
3. Accessibility checklist
