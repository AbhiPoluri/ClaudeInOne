# Content Good

## Invocation
`/content:good [request]`

## Purpose
Produce strategic, research-backed copy options.

## Agent Routing
- copywriter
- researcher

## Execution Protocol
1. Analyze audience + competing messaging.
2. Define messaging strategy.
3. Draft high-quality variants with rationale.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Strategy brief
2. Copy variants
3. A/B test suggestions
