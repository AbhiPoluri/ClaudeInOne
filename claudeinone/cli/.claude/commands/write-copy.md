# Write Copy

## Invocation
`/write:copy <brief>`

## Purpose
Generate conversion-oriented marketing copy options.

## Agent Routing
- copywriter

## Execution Protocol
1. Clarify offer and audience.
2. Draft multiple angles.
3. Recommend strongest CTA variant.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Copy variants
2. CTA options
