# Write Blog

## Invocation
`/write:blog <topic>`

## Purpose
Create SEO-aware blog post with strong structure.

## Agent Routing
- content-creator
- seo-specialist

## Execution Protocol
1. Build outline and target keyword intent.
2. Draft full article.
3. Add title/meta/internal linking suggestions.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Blog draft
2. SEO metadata
