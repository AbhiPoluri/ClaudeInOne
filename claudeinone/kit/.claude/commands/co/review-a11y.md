# Review Accessibility

## Invocation
`/review:a11y`

## Purpose
Accessibility audit for WCAG conformance.

## Agent Routing
- accessibility-auditor

## Execution Protocol
1. Check semantics/labels/focus flow.
2. Validate contrast and keyboard support.
3. Provide remediation guidance.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. A11y findings
2. Priority fixes
