# Video Script

## Invocation
`/video:script <topic>`

## Purpose
Generate structured video script with hook, body, and CTA.

## Agent Routing
- content-creator
- copywriter

## Execution Protocol
1. Define audience and desired outcome.
2. Draft hook/body/CTA script versions.
3. Add pacing and delivery cues.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Script variants
2. Recommended version
