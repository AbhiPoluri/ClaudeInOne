# Checkpoint

## Invocation
`/checkpoint`

## Purpose
Save session state and concise continuation notes.

## Agent Routing
- project-manager
- journal-writer

## Execution Protocol
1. Snapshot current progress.
2. Record unfinished work and blockers.
3. Provide resume instructions.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Checkpoint summary
2. Resume checklist
