# Docs README

## Invocation
`/docs:readme`

## Purpose
Create or update README for accurate setup and usage.

## Agent Routing
- docs-manager

## Execution Protocol
1. Audit current setup/run commands.
2. Update quickstart, scripts, architecture notes.
3. Verify examples against real commands.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. README update summary
2. Remaining unknowns
