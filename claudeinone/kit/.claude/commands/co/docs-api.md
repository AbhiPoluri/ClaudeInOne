# Docs API

## Invocation
`/docs:api`

## Purpose
Generate API reference docs from routes/controllers/schema.

## Agent Routing
- docs-manager
- backend-developer

## Execution Protocol
1. Discover endpoints and contracts.
2. Generate request/response docs.
3. Add auth/error examples.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. API docs files
2. Coverage gaps
