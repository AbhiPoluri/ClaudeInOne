# Load Skill

## Invocation
`/load <skill>`

## Purpose
Activate a named skill for current session context.

## Agent Routing
- skill-creator
- project-manager

## Execution Protocol
1. Resolve skill file by name.
2. Validate availability and compatibility.
3. Announce activated skill and scope.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Skill activation status
2. Applied behavior changes
