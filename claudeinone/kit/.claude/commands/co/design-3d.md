# Design 3D

## Invocation
`/design:3d [desc]`

## Purpose
Build Three.js/WebGL experiences with interaction and performance considerations.

## Agent Routing
- ui-ux-designer
- frontend-developer
- performance-optimizer

## Execution Protocol
1. Define 3D concept and scene goals.
2. Implement scene, lighting, assets, interactions.
3. Optimize draw calls/LOD and responsiveness.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Scene architecture
2. Implementation files
3. Performance checklist
