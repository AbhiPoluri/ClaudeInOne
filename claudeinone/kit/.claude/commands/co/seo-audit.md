# SEO Audit

## Invocation
`/seo:audit`

## Purpose
Run technical and content SEO audit.

## Agent Routing
- seo-specialist

## Execution Protocol
1. Check crawl/index fundamentals.
2. Audit metadata/content/internal links.
3. Prioritize SEO fixes by impact.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. SEO findings
2. Fix roadmap
