# Slide Create

## Invocation
`/slide:create <topic>`

## Purpose
Create slide deck structure with speaker notes.

## Agent Routing
- content-creator

## Execution Protocol
1. Define narrative arc and audience.
2. Create section-by-section slide outline.
3. Add concise speaker notes.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Slide outline
2. Speaker notes
