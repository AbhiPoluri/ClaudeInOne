# Brainstorm

## Invocation
`/brainstorm [idea]`

## Purpose
Generate alternative implementation ideas and tradeoff analysis.

## Agent Routing
- brainstormer
- researcher

## Execution Protocol
1. Frame the problem and constraints.
2. Produce multiple approaches.
3. Evaluate tradeoffs and failure modes.
4. Recommend a direction.

## Validation
- Confirm scope and constraints before making changes.
- Run relevant checks for each touched surface.
- Report unresolved risks or skipped validations.

## Output Format
1. Options matrix
2. Recommendation
3. Risks
