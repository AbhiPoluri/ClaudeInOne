# Plan CRO

## Invocation
`/plan:cro`

## Purpose
Plan conversion-rate optimization experiments.

## Agent Routing
- copywriter
- seo-specialist
- planner

## Execution Protocol
1. Audit current funnel/copy.
2. Identify high-impact CRO opportunities.
3. Propose experiment roadmap and metrics.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. CRO plan
2. Experiment matrix
3. Measurement plan
