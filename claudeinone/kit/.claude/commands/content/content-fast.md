# Content Fast

## Invocation
`/content:fast [request]`

## Purpose
Generate rapid copy variants for specific channel constraints.

## Agent Routing
- copywriter

## Execution Protocol
1. Identify format, audience, and tone.
2. Draft 3-5 variants quickly.
3. Include character-count-aware options.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Variant set
2. Recommended option by use case
