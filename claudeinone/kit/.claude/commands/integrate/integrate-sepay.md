# Integrate SePay

## Invocation
`/integrate:sepay [requirements]`

## Purpose
Implement SePay payment flows with localization and webhook security.

## Agent Routing
- integration-specialist
- backend-developer
- frontend-developer
- tester

## Execution Protocol
1. Implement QR and bank transfer flows.
2. Add signature-verified webhook handlers.
3. Apply Vietnamese localization/formatting constraints.
4. Validate end-to-end payment states.

## Validation
- Verify assumptions before changing files.
- Run the smallest relevant checks first, then broader checks when risk is high.
- Report residual risks explicitly.

## Output Format
1. Integration artifacts
2. Localization notes
3. Test results
